# འདིར་ཕེབས་པར་དགའ་བསུ་ཞུ།

ལས་རིགས་འདི་དང་འབྲེལ་བའི་ཆ་ཤས་ཡོངས་ལ་བལྟ་འདོད་ཚེ་དྲ་ཐག་འདི་ལ་གཟིགས།

## ལས་རིགས་རྣམ་བཞག

* `ལས་ཀའི་ངོ་བོ།
* `ལས་ཀའི་དམིགས་ཡུལ།
* `ལས་ཀ་དངོས་སུ་སྒྲུབ་ཐབས།
* `ལས་ཀའི་རམ་འདེགས།

## ལས་གཞིའི་ཞིབ་ཕྲ།

    ལས་ཀའི་དོ་དམ་སྡེ་ཁག
    ལས་གཉེར་བའི་འཚོ་བའི་སྐོར།
        ལས་ཀ་བའི་གླ་ཆའི་སྐོར།
        སྨན་རིན་སྐོར།
